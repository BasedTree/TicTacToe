Hello, Welcome to TicTacToe

You probably know the rules of tictactoe, but incase you dont:

Rules:
2 player game, each player takes turns placing an x or o in one of nine slots
whoever gets 3 in a row first (be it horizontally, vertically or on the diagonals), wins!


How to play: 
Run main.py, requires pygame

Versions used: 
pygame 2.5.1 
python 3.9.12)
